## Projects

> Each project folder contains:
> - `README.md` — Project documentation with GitHub repository link
> - `.zip` — Complete project zip file
> - Render deployment link *(for deployed projects)*

---

### 1. Adult Census Income Classification
Classification model to predict whether an individual earns more or less than $50K/year based on census data using machine learning algorithms.

---

### 2. Cancer Detection using MRI Images
An AI-powered cancer detection pipeline using **Multiple Instance Learning (MIL)** on MRI image patches to classify and detect cancerous tissue.

---

### 3. Cifar-10 Image Classification using CNN
Deep learning model built using Convolutional Neural Networks (CNN) to classify images into 10 categories from the CIFAR-10 dataset.

---

### 4. End to End Render Deployment Project
A complete end-to-end machine learning project covering model training, REST API development, and live deployment on **Render**.
*Render link available in project folder README.*

---

### 5. Face Recognition using CNN in Wild Life (LFW Dataset)
A face recognition system trained on the **Labeled Faces in the Wild (LFW)** dataset using CNN-based architectures for real-world identification tasks.

---

### 6. Movie Recommendation System
A personalized movie recommendation engine built using collaborative filtering and content-based filtering techniques.

---

### 7. RAG Chatbot - Capstone Project (Render)
A **Retrieval-Augmented Generation (RAG)** based intelligent chatbot deployed on Render, combining large language models with a custom knowledge base for context-aware responses.
*Render link available in project folder README.*
